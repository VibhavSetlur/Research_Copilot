# Research OS Skills Catalog

This document catalogs the research methodologies that the OS can execute via MCP tools.

## 1. Exploratory Data Analysis (EDA)
- **Description:** Automatically loads data, identifies data types, generates summary statistics, and visualizes distributions.
- **Related Tools:** `tool.eda.run`, `view.data.head`

## 2. Literature Review & Systematic Review
- **Description:** Searches academic databases, retrieves paper metadata, and generates summaries.
- **Related Tools:** `tool.pubmed.search`, `tool.semantic_scholar.search`, `tool.google.scholar.search`, `mem.literature.index`

## 3. Hypothesis Testing (Frequentist & Bayesian)
- **Description:** Executes statistical tests such as t-tests, ANOVA, and chi-square tests, including automatic assumption checking.
- **Related Tools:** `tool.statistical.test`

## 4. Causal Inference
- **Description:** Supports basic causal inference structures such as DAGs, Instrumental Variables (IV), Difference-in-Differences (DiD), and Regression Discontinuity Design (RDD).
- **Related Tools:** `tool.causal.infer` (Planned)

## 5. Predictive Modeling
- **Description:** Standard regression and classification models, as well as time-series forecasting.
- **Related Tools:** `tool.model.train`, `tool.model.predict`

## 6. Meta-Analysis
- **Description:** Aggregates findings across multiple branches or literature sources to form synthesized conclusions.
- **Related Tools:** `tool.meta.analyze`

## 7. Dashboard & Report Generation
- **Description:** Creates interactive HTML/Panel dashboards from experiment outputs.
- **Related Tools:** `tool.dashboard.create`, `tool.figure.create`

## 8. LaTeX Manuscript Compilation
- **Description:** Compiles LaTeX documents with bibliographies to generate publication-ready PDFs.
- **Related Tools:** `tool.latex.compile`
