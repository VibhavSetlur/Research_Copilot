# papers/ — Relevant Literature PDFs

Drop PDFs of papers relevant to your research here. The system will:

1. Extract titles, abstracts, and key findings
2. Map them to your research questions
3. Build an evidence matrix
4. Verify citations against CrossRef and Semantic Scholar

## Supported Formats

- `.pdf` — Full-text PDFs
- `.bib` — BibTeX files with citation metadata
- `.ris` — RIS export from reference managers

## Naming Convention

Use descriptive names: `smith_2024_income_satisfaction.pdf`, not `paper1.pdf`.

## What Gets Extracted

For each paper:
- Title, authors, year, DOI
- Abstract
- Key findings and effect sizes
- Sample size and population
- Methodology
- Limitations noted by authors

## Tips

- Include both supporting and contradicting evidence
- Add seminal papers in your field
- Include recent work (last 5 years) for currency
- The system will also search Semantic Scholar and PubMed automatically if < 20 papers are provided
