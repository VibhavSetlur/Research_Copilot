# Examples

This folder contains two fully runnable end-to-end examples designed to fit comfortably in GitHub.

- `beginner_study/`: a minimal single-analysis workflow.
- `expert_study/`: a branching workflow with synthesis.

## Run Both

From repository root:

```bash
./.venv/bin/python examples/beginner_study/run.py
./.venv/bin/python examples/expert_study/run.py
```

Each example writes outputs inside its own `outputs/` folder.
